# Ramé Editorial

Ramé Editorial nació en 2022 como un sueño compartido entre amantes de la literatura que creían que los libros no solo cuentan historias, sino que transforman vidas.

## Historia

El nombre **Ramé** surge de la combinación de raíces culturales y la pasión por las letras. Desde el inicio, nuestro propósito fue dar voz a nuevos autores y rescatar historias auténticas que conecten con los lectores.

## Misión

Promover la literatura y apoyar a escritores emergentes, ofreciendo un espacio donde las historias puedan crecer y llegar a más personas.

## Visión

Consolidarse como una casa editorial moderna e innovadora, comprometida con la calidad literaria y la difusión de nuevas voces.

## Sobre este proyecto

Este repositorio contiene el desarrollo del sitio web de **Ramé Editorial**, donde se presenta información sobre la editorial, sus autores y sus publicaciones.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Autor

Proyecto desarrollado para representar la identidad y visión de **Ramé Editorial**.
